# ComptaFrance Premium

Prototype React/Vite premium pour un site de comptabilité française.

## Installation

```bash
npm install
npm run dev
```

## Déploiement Vercel

- Framework: Vite
- Build Command: npm run build
- Output Directory: dist
- Install Command: npm install

## Fonctions incluses

- Design premium responsive
- Connexion démo front-end
- Dashboard comptable
- Factures clients
- Téléchargement facture HTML imprimable
- Export CSV des factures
- Module TVA
- Journal comptable
- Centre conformité

⚠️ Prototype front-end. Pour la production réelle, ajouter back-end, vraie authentification, base de données, PDF serveur, stockage sécurisé, journal immuable, export FEC conforme et validation par expert-comptable.
